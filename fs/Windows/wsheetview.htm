Getting Started

The Excel Viewer displays documents created in Microsoft Excel 97, Excel 2000 and Excel 2002.

The Excel Viewer can be opened by:

  1. Double clicking the Excel Viewer desktop icon.
  2. Clicking on the Start button, selecting Run and typing "Xls.exe" from the Run dialog.

The Zoom Feature

The Zoom feature allows you to enlarge or reduce the worksheet content displayed by the Excel Viewer. Zoom is available from the Toolbar or from the View menu.

To get a different view of a worksheet, select from the preset Zoom settings or choose a Custom Zoom from 50% to 200%.


Change Worksheets

Worksheets open with the first sheet showing in the viewer. To change sheets, simply click the desired worksheet from the Sheet option on the View menu.

From the keyboard, CTRL+ Page Down displays the next sheet, CTRL + Page Up displays the previous sheet.


Change Row Height and Column Width

To adjust the height of a row or the width of a column:

  • Right click on the Column Heading or Row Heading to bring up the Height/Width adjustment screen. Select the desired height or width.
  • Double-click the Column Heading or Row Heading to fit the data entry into one visible cell


Application Usage Notes

The Excel Viewer cannot display files that have been password protected using the "Save Options" password setting in Microsoft Excel.


Using the Splitter

You can Split a worksheet into Panes by selecting Split from the View menu. You can also Split a worksheet by dragging the Splitter from the upper right corner or lower left corner of the worksheet screen.

While the Splitter is open you can move around and view different sections of your worksheet by using the scroll bars. If you have an Action button on your device, you can use it to scroll within a pane.

To close the splitter, either select Remove Split from the View menu or drag the splitter to the upper right corner or lower left corner of the worksheet screen.


Move around in a Worksheet

Use the Scroll Bars:

When the worksheet data is larger than one screen, scroll bars appear to enable you to move around.

  • Click the up/down and right/left arrows to move one row or one column in the desired direction.
  • Click the space between the arrow and the scroll bar to move one screen in the desired direction.

Using an External VGA

If you have an External Video Adapter for your device, you can configure the Excel Viewer to display on an external viewer. You can set up the adapter by doing the following:

  1. Insert the adapter in your device.
  2. From the Excel Viewer select Preferences from the Setup menu.
  3. Set the following options from the Adapter dialog:
    • The "Use external video adapter" checkbox will be enabled when the adapter has been detected.
    • If you have more than one adapter, you can select your adapter from the Adapter drop down list box.
    • The Video mode lists the video options in the following order: Height, Width, Color (by bits) and Refresh rate.
    • If your card supports multiple outputs, they will be listed in the Output to drop down list.

While the document is showing on your External VGA, you can navigate through the document from your device.


Printing

  1. From the Excel Viewer, select Print from the File menu. The Print dialog window will be displayed.
  2. Select the printing options you want.

Notes
  • Port lists the available printer ports (i.e. COM1, IRDA, Network).
  • To print to a network printer, select the network option from the Port list and enter the path to the network printer in the Net Path.
