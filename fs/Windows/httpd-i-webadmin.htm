What is WebAdmin?

As discussed in the overview, server-side scripts, such as WebAdmin, are used to remotely configure your Windows CE device using your Web browser and the Web server running on the device. WebAdmin is used to configure the settings related to the device's Web server itself.

There are a number of reasons why you may want to change the settings of the Web server. You may want to set up a directory that holds pictures of your family to be shared with everyone on the Internet. (See virtual directory configuration for more information.) Or you may want to set up the server so that you and your spouse have access to certain data on the server and your children have access to a more restricted set of data. (See authentication overview for more information.)

Caution: If you misconfigure your device using WebAdmin, you can open yourself up to attacks by hackers and/or get into a state where you lock yourself out of your device. If you do not understand a configuration setting in WebAdmin, do not change it.
