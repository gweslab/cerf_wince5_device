Using WebDAV to upload files.

The Windows CE Web Server supports WebDAV, which is a protocol that enables web clients to upload files. On Windows XP and other Microsoft products, WebDAV is sometimes referred to as "Web Folders".

To allow a Web server virtual directory to act as a Web Folder, you will need to take the following steps on the server:

  • Carefully consider the security implications of this action.
  • Enable WebDAV globally for the Web site.
  • Set the virtual directory to allow directory browsing. Click here for instructions.
  • Set the virtual directory to grant write permissions. Click here for instructions.
  • Restart the Web server to make the changes take effect.

Now you need to configure your client machine to use the WebDAV folder you have shared. On Windows XP, click on the Start button, select Run, and then type \\WinCEDeviceName\DavFolder. WinCEDeviceName is the device's name and DavFolder is the name of the folder you have shared. After running this command, a browser window will appear that will let you browse and modify the virtual root as if it were a directory on your hard drive. To use WebDAV with other operating systems and applications, check your documentation.

Caution: If a hacker can gain write access a WebDAV share, they can add or delete files to your device. Always require authentication on WebDAV shares that are not on trusted networks. $$else$$
