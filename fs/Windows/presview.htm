Getting Started

The PowerPoint Viewer displays presentations created in Microsoft PowerPoint 97, PowerPoint 2000 and PowerPoint 2002.

The PowerPoint Viewer allows you to view your presentation as it was created or to create a Custom Slide List. In this list, you can select which slides to include in the presentation and you can change the order of the slides presented.

Use a Pen to mark or accent a slide during a presentation.

The PowerPoint Viewer opens displaying the first slide in the presentation in either the Normal View or the Slide Sorter View. From the current View you can Change the View, Set Up a Show, Save the Show Settings or View your show. From the Normal View window you can also Zoom in on a slide.

The PowerPoint Viewer can be opened by:

  1. Double clicking the PowerPoint Viewer desktop icon.
  2. Clicking on the Start button, selecting Run and typing "PresViewer.exe" from the Run dialog.

View a Presentation

View Show is available from the Slide Show menu or the View Show ToolBar icon.

If your presentation contains a PowerPoint Custom Show, you can select it from the Custom Show List on the menu bar. Or, you can change the Custom Slide List, by selecting which slides to include and changing the order of the slides in your presentation.

The Set Up Show dialog allows you to set Slide Transition timings.

While a show is running, you can move to another slide by using one of the Navigation techniques.

A show can be interrupted by right clicking (click on the Watermark) on the presentation screen and choosing End Show from the Context menu.


Change the View

The PowerPoint Viewer opens with the presentation in the Normal View. You can change to the Slide Sorter by selecting Slide Sorter from the View menu or by using the Slide Sorter ToolBar Icon.


View Notes

If your presentation contains Notes, you can view the notes from the Normal View window.


Edit a Custom Slide List

The PowerPoint Viewer allows you to Select which slides to include in your presentation, Sort the slides as desired and to set the Slide Transition Timings.

  • You can Sort, Select and Hide slides from the Custom Show Window. To display the Custom Show Window, select Slide Show from the Main window then select the Custom Show option.
  • Slide transition timings can be set in the Set Up Show dialog.

When your show is complete, you can Save the show settings.


Navigating a Presentation

While a show is running, you can move to another slide by doing one of the following:

  • Click on the presentation screen to advance one slide forward. NOTE: This is disabled when using a Pen.
  • While a presentation is running, click on the arrow watermark in the lower left corner of the screen to bring up a Context menu. From this menu, you can view the Next slide action, the Previous slide action or select a slide from the Go to... drop down list.
  • Use the Cursor or Action button on your device to move to the Next or Previous slide. The Right and Down buttons move to the Next slide and the Left and Up buttons move to the Previous slide.

The Normal View

The Normal View has three panes, the Slide List, the Image Window and the Notes Window.

The Slide List window allows you to do the following:

  • Choose the image shown in the Image window by selecting the slide in the list.
  • Move a slide up or down by using the mouse to drag and drop the slide to the desired location.

The Image Window shows a preview image of the currently selected slide.

From the Image Window you can do the following:

  • Scroll through the slides using the scroll bars.
  • Zoom in on a slide for a closer view.

    If your presentation contains a PowerPoint Custom Show, you can select the show from the Custom Show List on the menu bar.


    Using a Pen

    While a presentation is running, you can use a Pen to mark on a slide. The Pen is available from the Context menu obtained by tapping on the arrow watermark on the lower left corner while a presentation is running.

    While a show is running, you can move to another slide by using one of the Navigation techniques.


    Save Show Settings

    After creating a Custom Show you can Save your show settings. These settings will automatically load when you reload the show. To return to the original PowerPoint Presentation settings, select Default Settings from the File menu after the show has loaded.


    The Slide Sorter

    The Slide Sorter Image Window shows slide thumbnails of the currently selected presentation. From this window, you can do the following:

    • Sort the slides by using the mouse to drag and drop the slide to the desired location.

    If your presentation contains a PowerPoint Custom Show, you can select the show from the Custom Show List on the menu bar.


    Set Up a Show

    The Set Up Show dialog can be opened from the Slide Show menu. From the Set Up Show dialog, you can do the following:

    • Set the Slide Transitions advance method to Manual, Use timings or Automatic.
    • Set the Screen Rotation for a Slide Show to Landscape or Portrait.
    • Select whether or not to set to end the Slide Show with a black slide.

    Set Up Slide Transitions

    From the Set Up Show dialog you can select how the show displays.

    • The Use timings, if present button will display the show just as it does in PowerPoint.
    • The Manually button will ignore all PowerPoint timings, allowing you to control the time between animations and transitions.
    • The Automatically advance after button will allow the show to run on its own. When the Automatic buttons is selected, you can set the Timing between slide transitions and Loop the show if desired.

    End the Show:

    • Automatic Slide Transitions can be interrupted by right clicking on the presentation (click on the Watermark) and choosing End Show from the Context menu.

    Zoom

    For a close-up view of a slide, you can Zoom by selecting the desired magnification from the Zoom List while in the Normal View.


    Using an External VGA

    If you have an External Video Adapter for your device, you can configure the PowerPoint Viewer to display on an external viewer. You can set up the adapter by doing the following:

    1. Insert the adapter in your device.
    2. From the PowerPoint Viewer select Preferences from the Setup menu.
    3. Set the following options from the Adapter dialog:
      • The "Use external video adapter" checkbox will be enabled when the adapter has been detected.
      • If you have more than one adapter, you can select your adapter from the Adapter drop down list box.
      • The Video mode lists the video options in the following order: Height, Width, Color (by bits) and Refresh rate.
      • If your card supports multiple outputs, they will be listed in the Output to drop down list.

    While the presentation is showing on your External VGA, you can use the Pointer and Pen to mark or accent items on your display. You can Navigate through the presentation from your device.


    Printing

    1. From the PowerPoint Viewer, select Print from the File menu. The Print dialog window will be displayed.
    2. Select the printing options you want.

    Notes
    • Port lists the available printer ports (i.e. COM1, IRDA, Network).
    • To print to a network printer, select the network option from the Port list and enter the path to the network printer in the Net Path.
