Getting Started

You can use the Image Viewer, to view and edit BMP, JPEG and PNG images.

The Image Viewer allows the user to control how images are viewed in order to create a Slide Show. Images can be Sorted, Rotated and Zoomed. The Timings between slides can be set and there are a variety of Slide Transition Effects to choose from.

The Image Viewer Edit Window allows you to adjust the Brightness/Contrast and Hue/Saturation of an image. You can also Scale, Crop, and change the Orientation of an image, then save it as a JPEG file.

The Image Viewer can be opened by:

  • Double clicking the Image Viewer desktop icon.
  • Clicking on the Start button, selecting Run and typing "ImageViewer.exe" from the Run dialog.

Hide an Image in a Show

You can Hide an image so it does not display in your Slide Show. Hide Image is available from:

  • Thumbnail View: Highlight the image thumbnail, go to the Slide Show menu and select Hide Image. A red X will appear on the slide to indicate it will not be shown in the slide show.
  • Context Menu: Hide Image is also available by right clicking on a thumbnail image and selecting Hide Image from the context menu.
  • Set Up Show Dialog: Hide the image by deselecting the checkbox in the Set Up Show dialog.
  • To unhide the image, deselect the Hide Image in one of the menus or in the Set Up Show dialog.

Locate the Images

To locate images stored on your device, select Browse... from the File menu. The View Files in... dialog shows a tree view of the directory structure on your device. You can browse the tree to locate your image files.

You can also type the path to your images from the Location bar just below the menu bar. If your device is on a network, typing the network path in the Location bar will allow access to networked images.


Open an Existing Slide Show

To open a previously saved show, choose Open Show Settings from the File menu. From the Open dialog, you can browse the directory tree to locate your Show Settings file. The file extension for a Show Settings file is .wss.


Image Orientation

Images can be Rotated and Flipped in a slide show. Choose one of the following options to change the orientation of an image:

  • Full Screen View: Open an image to Full Screen View. Right Clicking on the image brings up a Context menu (click on the Watermark for Handheld PC) where you can select the Image Orientation from the Rotate submenu.
  • Set Up Show Dialog: Image Orientation can also be set from the Set Up Show dialog.

Save the Show Settings

After Creating a Slide Show, you can save the Show Settings. From the File menu, select Save Show Settings As... to save the show. Once a show has been saved, you can choose Save Show Settings to save the show or Save Show Settings As... to choose the directory or a new name for the Slide Show settings file.


Set Up a Slide Show

The Set Up Show dialog is available from the Slide Show menu or from the Set Up Show ToolBar icon.

  • To change the settings for one slide in the show, select the slide in the Slide List and change the Image Settings at the bottom of the Set Up Show dialog.
  • To change the settings for all slides in the show, use the Select All button.
  • The Checkbox in the Slide List allows you to remove slides from a Slide Show.
  • After setting up a show, you can Save the Show Settings.

Tip: It is easier to set the Zoom settings in the Full Screen View before Setting Up the Show.


Sort Images

You can choose the order you wish your images to display. Ordering the thumbnail images can be done by:

From the Edit menu you can select Sort Images to choose sort by Name, Type, Size or Date. Submenus allow you to sort in Ascending or Descending order.

Custom Sorts are available with the following options:

  • Drag & Drop: Click on a thumbnail and drag it to the location in the Thumbnail View where you wish it to display.
  • ToolBar icon: Select an image and use the Move Selected Image ToolBar icons to move the image up or down in display order.
  • Set Up Show: You can use the Move Selected Image ToolBar icon in the Set Up Show dialog to change the order of the images.

Slide Transition Effects

There are over 30 different Slide Transitions available. Slide Transitions are set in the Set Up Show dialog.


View a Single Image

To View a single image, you can double click the image name in Windows Explorer.

Right Clicking on the image will give you a context menu (click on the Watermark for Handheld PC) with the options to Zoom or Rotate the image.

A single image can also be Viewed in the Image Viewer.


View a Slide Show

After Creating a Slide Show, you can view and preview the show. View options include Manual Advance, Automatic Advance and Automatic Advance (Loop). You can begin your Show by selecting View Show from the Slide Show menu or from the View Show ToolBar icon.

View Image Information

The Information Window gives details about File Size, Image Type, Date Modified, Date Taken, Color and Image Size. To access the Information Window, first select a Thumbnail in the Thumbnail Image Screen, then open the dialog from one of the following options:

  • From the View Menu: You can open the View menu and select Information Window.
  • From the ToolBar icon: Use the pointer to click on the Information Window ToolBar icon.

To deselect the Information Window, use any of the above options to toggle it off.

Thumbnail View vs. Full Screen View

Thumbnail View:

  • The Image Viewer opens showing the images in the Thumbnail View. This is the default setting when opening images in the Image Viewer.
  • From the Thumbnail View you can see all the slides in your current slide directory. You can use the menus to save your show, sort your images, view the Information Window, Set Up your show and View your show.
  • The ToolBar icons allow you to Order the images, view the Information Window, go to Full Screen View, Set Up your show and View your show.

Full Screen View:

  • Full Screen View allows an image to be enlarged to fill the screen of your device. You can access Full Screen View by double clicking a thumbnail, right clicking a thumbnail, selecting Full Screen from the View menu or by using the View Full Screen ToolBar icon.
  • From Full Screen View, you can right click on an image to bring a Context menu (click on the Watermark) where you can set the Zoom and Rotate options.
  • Clicking on an image while in the Full Screen View will display the next image in your show sequence.

Zoom to Resize the Image

Zoom allows you to reduce or enlarge an image displayed by the Image Viewer. Valid settings for Zoom are from 25% to 200%, "Fit to Width" and "Fit to Window".

Zoom is set in the following ways:

  • Full Screen View: Open an image to the Full Screen View and right click on the image to bring up a Context menu (click on the Watermark). You can select your Zoom settings from the Zoom sub-menu.
  • Set Up Show Dialog: Zoom can also be set from the Set Up Show dialog.

Using an External VGA

If you have an External Video Adapter for your device, you can configure the Image Viewer to display on an external viewer. You can set up the adapter by doing the following:

  1. Insert the adapter in your device.
  2. From the Image Viewer select Preferences from the Slide Show menu.
  3. Set the following options from the Adapter dialog:
    • The "Use external video adapter" checkbox will be enabled when the adapter has been detected.
    • If you have more than one adapter, you can select your adapter from the Adapter drop down list box.
    • The Video mode lists the video options in the following order: Height, Width, Color (by bits) and Refresh rate.
    • If your card supports multiple outputs, they will be listed in the Output to drop down list.

While the slide show is showing on your External VGA, you can navigate through the slide show from your device.


Printing

  1. From the Image Viewer, select Print from the File menu. The Print dialog window will be displayed.
  2. Select the printing options you want.

Notes
  • Port lists the available printer ports (i.e. COM1, IRDA, Network).
  • To print to a network printer, select the network option from the Port list and enter the path to the network printer in the Net Path.


Edit Window

The Image Viewer Edit Window allows you to adjust the Brightness/Contrast and Hue/Saturation of an image. You can also Scale, Crop, and change the Orientation of an image, then save it as a JPEG file.

The Edit Window can be opened by clicking Edit Image... from the Edit menu. Then click on the Edit menu to display the Edit Control options available.

Zoom

Once the Edit Window is open, the default zoom setting for the image is Fit to Window. The default setting can be changed by clicking on the View menu and selecting a different zoom option.

Brightness and Contrast

To adjust the Brightness and Contrast of the image, simply slide the bar to the left and right until you get the desired result. The Undo button will clear any changes made from when the current edit pane was opened.

Hue and Saturation

The Hue and Saturation option allows you to change the tone of the color and the amount of color present in an image. To adjust the Hue and Saturation of an image, simply slide the bar to the left and right until you get the desired result. The Undo button will clear any changes made from when the current edit pane was opened.

Scale

The Scale option allows you to resize the image by selecting the percentage of current size from the drop down list or adjusting the Width and Height of the image in the edit boxes. Select the Apply button to view the changes. The Undo button will clear any changes made from when the current edit pane was opened.

Crop

The Crop option allows you to crop an image by dragging over the area you want to keep or adjusting the crop area with the Left, Right, Top, and Bottom edit boxes. Select the Apply button to view the changes. The Undo button will undo any changes made from when the current edit pane was opened.

Audio

If your device has recording capabilities, the Audio option allows you to add a Voice-Recorded Message to your image. To record a message, select the Record icon and when finished recording select the Stop icon. To play back the recorded message select the Play icon. To erase the message select the Clear button. After the image has been saved, the message recorded is played back during a Slide Show.

Image Orientation

You can change the Orientation of an image by selecting Rotate Right, Rotate Left, Flip Horizontal, and Flip Vertical from the Edit menu.

Save As

The Save As option from the File menu allows you to save the changes you have made in the Edit Window as a JPEG file. If you exit the Edit Window without saving the changes, your changes will not be saved.

Revert to Saved

If you make changes to your image and choose not to save them, you may revert back to the last saved image. To revert back to the last saved image click File then Revert to Saved to load the last saved image into the Edit Window.
