Getting Started

The Word Viewer displays documents created in Microsoft Word 97, Word 2000 and Word 2002. It also displays documents saved in Rich Text Format (.rtf).

The Word Viewer can be opened by:

  1. Double clicking the Word Viewer desktop icon.
  2. Clicking on the Start button, selecting Run and typing "DocViewer.exe" from the Run dialog.

Moving Around in a Document

When a document is larger than one screen, scroll bars appear to enable you to move around.

  • Click the up/down and right/left arrows to move one line in the desired direction.
  • Click the space between the arrow and the scroll bar to move one screen in the desired direction.

You can also use the Page Up or Page Down keys to move one screen. The left, right, up and down Arrow keys will move you one line in the desired direction.


The Zoom Feature

The Zoom feature allows you to enlarge or reduce the Document content displayed by the Word Viewer. Zoom is available from the Toolbar or from the View menu.

To get a different view of a Document, select from the preset Zoom settings or choose a Custom Zoom from 25% to 200%.


Using an External VGA

If you have an External Video Adapter for your device, you can configure the Word Viewer to display on an external viewer. You can set up the adapter by doing the following:

  1. Insert the adapter in your device.
  2. From the Word Viewer select Preferences from the Setup menu.
  3. Set the following options from the Adapter dialog:
    • The "Use external video adapter" checkbox will be enabled when the adapter has been detected.
    • If you have more than one adapter, you can select your adapter from the Adapter drop down list box.
    • The Video mode lists the video options in the following order: Height, Width, Color (by bits) and Refresh rate.
    • If your card supports multiple outputs, they will be listed in the Output to drop down list.

While the document is showing on your External VGA, you can navigate through the document from your device.


Printing a Document

  1. From the Word Viewer, select Print from the File menu. The Print dialog window will be displayed.
  2. Select the printing options you want.

Notes
  • Port lists the available printer ports (i.e. COM1, IRDA, Network).
  • To print to a network printer, select the network option from the Port list and enter the path to the network printer in the Net Path.
